# BETH Stack Demos

This repo experiments with the BETH stack:

* *Bun* TypeScript runtime
* *Elysia.js* Minimalist HTTP Server library
* *Turso* SQLite Edge Database
* *HTMX* Minimalist client-side interaction

## Directions

1. Install Bun
2. Install Bun Dependencies

```cmd
bun install
```

3. Run the app

```cmd
bun dev
```
